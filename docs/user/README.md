## نکته

بعضی فایل‌های حسابداری در حال حاضر برای توسعه‌دهنده‌ها هم نوشته شده‌اند و ممکن است نمونه کد یا ساختار دیتابیس داشته باشند. برای استفاده روزمره، روی تعریف مفاهیم، مثال‌های عددی و نتیجه ثبت‌ها تمرکز کنید.
