معرفی پروژه

حسابداری آزاد برای نیازهای واقعی کسب و کار ایرانی

امیر یک نرم افزار حسابداری آزاد و رایگان مبتنی بر Laravel است. تمرکز پروژه روی ثبت های دوبل، فاکتور فروش و خرید، مدیریت کالا، عملیات سال مالی و تجربه کاربری فارسی است.

پروژه هنوز در مسیر توسعه فعال قرار دارد و پیش از استفاده عملیاتی باید با داده آزمایشی، نیازهای هر مجموعه و کنترل های حسابداری مستقل بررسی شود.

ویژگی ها

قابلیت های اصلی امیر

حسابداری و گزارش

مدیریت درآمد و هزینه، فاکتور و رسید، گزارش های مالی و پایه های لازم برای قواعد مالیاتی ایران.

انبارداری و کالا

مدیریت محصولات، موجودی، گردش انبار و محاسبه بهای تمام شده با میانگین موزون متحرک.

سال مالی

ایجاد، نگهداری، خروجی گرفتن و وارد کردن داده های سال مالی با توجه به مرزهای حسابداری.

حضور و غیاب

ثبت ورود و خروج، پیگیری کارکرد روزانه و ماهانه و ورود اطلاعات از منابع خارجی.

حقوق و دستمزد

محاسبه حقوق ماهانه، مدیریت کسورات و مزایا، فیش حقوقی و گزارش های مرتبط.

متن باز و قابل توسعه

کد آزاد با معماری سرویس محور Laravel برای مشارکت، بررسی و توسعه شفاف.

شروع سریع

سه مسیر نصب، از آزمایش سریع تا محیط عملیاتی

راهنمای نصب، Docker Compose، اجرای تک دستوری Docker و نصب استاندارد با PHP و MariaDB را پوشش می دهد.

مشاهده راهنمای نصب
۱

Docker Compose برای محیط عملیاتی

استفاده از ایمیج های آماده GitHub Container Registry با فایل تنظیمات محیطی.

۲

Docker تک دستوری برای آزمایش

راه اندازی سریع یک کانتینر همه در یک برای ارزیابی اولیه و تست کوتاه مدت.

۳

نصب استاندارد Laravel

راه اندازی با PHP، Composer، Node.js و MariaDB برای توسعه یا استقرار سفارشی.
