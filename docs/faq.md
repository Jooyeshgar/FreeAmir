# سوالات متداول


## آیا امیر واقعا رایگان است؟

بله. امیر تحت مجوز GPL-3 منتشر شده است. استفاده، تغییر و توزیع آن کاملا رایگان است. هیچ هزینه مجوز یا اشتراکی وجود ندارد.


## آیا می‌توانم آن را روی سرور خودم میزبانی کنم؟

بله. می‌توانید آن را با Docker، Docker Compose یا نصب استاندارد PHP و MariaDB روی سرور خودتان راه‌اندازی کنید. داده‌های شما کاملا در اختیار خودتان هستند.


## آیا از سامانه مودیان پشتیبانی می‌کند؟

بله. امیر از سامانه مودیان پشتیبانی می‌کند. هر شرکت می‌تواند گواهی و کلید خصوصی خود را تنظیم کند. برای راه‌اندازی [راهنمای سامانه مودیان](moadian.md) را مطالعه کنید.


## آیا از چند شرکت پشتیبانی می‌کند؟

بله. امیر از چند شرکت به صورت همزمان پشتیبانی می‌کند و هر شرکت می‌تواند چندین سال مالی مجزا داشته باشد.


## آیا از حقوق و دستمزد پشتیبانی می‌کند؟

بله. امیر شامل ماژول حقوق و دستمزد است که محاسبه حقوق ماهانه بر اساس حضور و غیاب، مدیریت کسورات و مزایا، صدور فیش حقوقی و تهیه گزارش‌های مرتبط را پوشش می‌دهد.


## آیا آماده استفاده عملیاتی است؟

امیر در حال توسعه فعال است و هنوز برای استفاده عملیاتی بدون بررسی و تست مستقل توصیه نمی‌شود. پیش از استفاده در محیط Production، بخش‌های مورد نیاز خود را با داده آزمایشی بررسی کنید.


## چه کسی می‌تواند در توسعه امیر مشارکت کند؟

هر کسی! امیر یک پروژه متن باز است و از مشارکت همه استقبال می‌شود. برای اطلاعات بیشتر [راهنمای مشارکت](https://github.com/Jooyeshgar/FreeAmir/blob/main/CONTRIBUTING.md) را مطالعه کنید.


## چه فناوری‌هایی استفاده شده است؟

- **بک‌اند:** Laravel 12 (PHP)
- **دیتابیس:** MariaDB
- **فرانت‌اند:** Tailwind CSS + Alpine.js
- **استقرار:** Docker / Docker Compose
- **مجوز:** GPL-3.0
