# Amir vs Commercial Software

As an open-source and free alternative, Amir offers compelling advantages over commercial accounting software.

---

## Feature Comparison

| Feature | Amir | Commercial Software |
|---|---|---|
| **Open Source** | ✅ Source code available | ❌ Closed |
| **Self-Hosted** | ✅ Run on your own server | ⚠️ Vendor-dependent |
| **Multi-Company** | ✅ Full support | ⚠️ Usually separate licenses |
| **Moadian** | ✅ Full support | ⚠️ Often paid add-on |
| **Payroll** | ✅ Includes attendance | ⚠️ Usually separate |
| **Inventory** | ✅ Weighted average costing | ⚠️ Varies |
| **License Cost** | ✅ Free (GPL-3) | ❌ Annual subscription |

---

## Key Advantages

### Freedom & Data Ownership

- Source code fully accessible and auditable
- Data stored on your own servers
- No vendor lock-in
- Customizable for specific needs

### Cost

- No license fees
- No annual subscriptions
- No user count limits
- No company count limits

### Transparency

- All financial calculations auditable
- Accounting standards followed
- Community contribution possible

---

## Alternative to Commercial Software

Amir can serve as an open-source, free alternative to commercial Iranian accounting software. Amir focuses on:

- Standard double-entry accounting
- Compliance with Iranian tax regulations
- Native Persian user interface
- Self-hosted deployment capability
