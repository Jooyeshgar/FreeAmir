# نقشه راه امیر

امیر یک پروژه در حال توسعه است. در اینجا برخی از ایده‌ها و اولویت‌های آینده فهرست شده‌اند.

---

## در حال توسعه

- نقش‌ها و دسترسی‌های اختصاصی برای هر شرکت

---

## ایده‌های آینده

### نقش‌ها و دسترسی‌ها
- نقش‌ها و دسترسی‌های اختصاصی برای هر شرکت

### فرآیندهای تأیید
- سیستم تأیید مراحل مختلف عملیات مالی

### گزارش‌های حسابرسی
- ثبت تغییرات و پیگیری فعالیت‌ها

### گسترش API
- API کامل برای اتصال به سیستم‌های جانبی

### پشتیبانی موبایل / PWA
- اپلیکیشن موبایل یا PWA برای دسترسی آسان‌تر

### گزارش‌های پیشرفته
- گزارش‌های مالی و مدیریتی پیشرفته‌تر
